# Setup
Download this repository

```console
git clone https://github.com/radsi/hackaclub-discord-bot.git
```

Then create a .env file and set your bot token and hackclub api key

```
BOT_TOKEN=
HACKCLUB_API_KEY=
```

Then run Base/app.js

```console
node Base/app.js
```
